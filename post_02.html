TASK
Produce multimedia projects using an interactive documentary platform and explore the definition of “interactive documentary” itself
OVERVIEW
“Global Digital Cities(2016)” was hosted by Studio for International Media & Technology, The University of Salford, MediaCityUK. Five international artists teams collaborated to how to escape linearly edited video format and to interactively engage viewers in creating and navigating the narrative. Following are the three sub-topics:

WORKFLOW
LEVEL DESIGN
Each city can have several scenes, each scene are tagged, and each theme also has tags. All the levels can be chosen as a sorting standard in the graph.
SCENE & SCENE EDITOR

Each city has several scenes. "SCENE" is the core of this interactive documentary. Scene is micro-unit documentaries which is a set of media data and can be grasped with in few seconds. Sound, video, image, and texts comprise each scene.

Each scenes are annotated with tags, which viewers can select with the graphic user interface to be shown on the final installation. The mood, dynamics of this small unit can be adjusted in JSON format. Tags can also be annotated here.
